# GitHub profile viewer

This is a simple dashboard that displays information about a GitHub's user
account.

## Features

* Show the user's avatar, bio, follower counts, contact metadata
* Repository analyics - stars, forks, and most used languages
* LocalStorage caching: cache searched results for 10 minutes to avoid GitHub's
public API rate limiting, with live synchronization if needed.
* Input Sanitation - Sanitize input usernames before searching
* Search History - store the last 5 searches as clickable bookmarks

## Project Structure

The codebase is organized as a component-driven architecture with clear separation of structural concerns:
# DevPulse Inspector 🔍🧬

DevPulse Inspector is a high-fidelity, client-side dashboard that uncovers a GitHub developer's core DNA, coding habits, and repository statistics. Built using modern React and styled with a sleek dark-mode user interface, it provides deep analytical footprint insights by parsing profile metadata and historical project repositories.

---

## ✨ Features

*   **Developer DNA Profiling:** Visualizes avatar, bio, follower counts, and contact metadata with graceful structural text fallbacks for incomplete public profiles.
*   **Repository Analytics:** Loops through up to 100 recent projects to calculate total accumulated stars, total forks made, top three most frequently used languages, and an original project vs. fork ratio.
*   **Smart `localStorage` Caching:** Bypasses aggressive public GitHub API rate limiting by caching payloads locally for 10 minutes.
*   **Live Synchronization:** Features an explicit "Sync Live Data" force-refresh switch to flush custom cache blocks instantly.
*   **Input Sanitization & Constraints:** Sanitizes input in real-time, matching validation against alphanumeric regex targets while safely honoring mixed-case lookups.
*   **Historical Sticky Navigation:** Maintains a running search log of your last 5 unique queries as clickable bookmarks with asynchronous action spam preventions.
*   **Animated Shimmer Skeletons:** Implements purely style-driven layout skeletons to transition smoothly during live network fetches.
